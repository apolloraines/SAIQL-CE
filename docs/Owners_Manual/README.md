# SAIQL Owners Manual

Welcome to the official **SAIQL Owners Manual**. This guide is designed to take you from a fresh Ubuntu installation to running a production-grade Semantic AI Query Language (SAIQL) system.

**SAIQL** is a unified database engine that bridges the gap between traditional SQL and modern query patterns. It operates in three distinct modes:
1.  **Standalone Engine**: A fully independent database server.
2.  **Translation Layer**: A semantic interface that translates natural language/SAIQL into SQL for legacy databases (Postgres/MySQL).
3.  **Migration Tool**: A bridge to import data from legacy systems into SAIQL's native storage.
